# System Automation Scripts

## Overview
This repository contains a collection of Python and Bash scripts developed to practice and automate common system administration and operational tasks. These scripts focus on improving efficiency, reducing manual work, and reinforcing scripting fundamentals.

## Purpose
The goal of this repository is to build practical scripting skills relevant to IT support and cybersecurity, including automation of repetitive tasks and basic system operations.

## Technologies Used 
- Python
- Bash (Shell scripting)

## Script Categories

### Account Management 
- create user accounts
- disable or modify existing accounts
- automate user-related administrative tasks

### Automation & Operations
- execute remote commands
- perfrom routine system operations
- automate repetitive workflows

### Security & Utilities 
- password generators
- random number generators
- basic scripting for secure practices

### Communication
- automated email scripts
- notification-based automation

## Example Use Cases
- automating onboarding/offboarding of user accounts
- generating secure passwords for users or systems
- running commands across systems remotely
- sending automated notifications or alerts
